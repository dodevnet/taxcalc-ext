# 税金计算工具

含税金额与不含税金额的快速换算浏览器扩展（Chrome / Edge / Brave 等 Chromium 内核浏览器），支持常见税率预设与自定义税率。

## 功能

- **两种计算模式**：含税 → 不含税 / 不含税 → 含税，一键切换
- **税率预设**：1%、3%、6%、9%、13%，下拉即选
- **自定义税率**：支持任意小数税率（如 1.5%、5.2%）
- **实时计算**：输入即出结果，无需手动点击
- **数值复制**：点击结果区任意金额一键复制到剪贴板
- **浮点精度保障**：使用 `EPSILON` 修正法确保保留两位小数的四舍五入结果准确

## 安装

### 开发者模式加载（本地）

1. 克隆仓库或下载源码
2. 打开浏览器扩展管理页：
   - Chrome/Edge：地址栏输入 `chrome://extensions`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目目录
5. 点击工具栏上的扩展图标即可使用

## 使用方式

1. 点击浏览器工具栏的扩展图标打开弹出面板
2. 选择计算模式：「含税 → 不含税」或「不含税 → 含税」
3. 输入金额
4. 选择或输入税率
5. 结果实时展示：**目标金额**、**原金额**、**税额**
6. 点击任意结果数值即可复制

## 计算逻辑

| 模式 | 公式 |
|------|------|
| 含税 → 不含税 | `不含税 = 含税 / (1 + 税率)` |
| 不含税 → 含税 | `含税 = 不含税 × (1 + 税率)` |
| 税额 | `税额 = 含税 - 不含税` |
